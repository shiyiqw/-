回来看代码或者配环境的时候，盯着这几个版本，别弄错了：

重要：项目用的是 BepInEx 6.0 版本，千万别去官方仓库顺手下了 5.0。官方默认主页一般是 5.0，6.0 的版本需要去他们的 Releases 列表里单独找一下。

2. Harmony
配合 BepInEx 6.0 一起用，用来处理运行时的方法拦截和 Hook。

3. MinHook (C++)
用的是最新版本，直接用 vcpkg 下载安装最省事：vcpkg install minhook。

4. 翩翩的 AI 声音模型
模型目前已经完全训练好了，等后面新剧情代码写完，直接对接进去用作配音。
